# ЛР1 — Монады (Вариант 9: Склад заказов)

Проект демонстрирует:
- собственные `Monad`, `Reader`, `Writer`, `State`, `IO`;
- вынос конфигурации в `Reader`;
- логирование бизнес-этапов через `Writer`;
- изменение состояния склада через `State`;
- консольный сценарий через собственный `IO`.

## Структура

- `src/main/scala/monads` — инфраструктура монад (Блок 0);
- `src/main/scala/domain` — предметные типы и бизнес-логика;
- `src/main/scala/app` — сценарий взаимодействия с пользователем (`IO`);
- `src/test/scala` — обязательные тесты по кейсам задания.

## Запуск

```bash
sbt run
```

Запуск тестов:

```bash
sbt test
```
